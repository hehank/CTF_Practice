# 概述
- 使用單個密鑰來加密(Encrypt)和解密(Decrypt)
    ![](https://i.imgur.com/cvjcQHr.png)

# 優點
- 速度快
- 需要較少計算資源

# 缺點
- 如果密鑰再傳送過程中被截取，截取者可以輕易解密

# 長度
- 通常為128或256 bit (取決於所需的安全級別)
